# Tecnoteca - Plataforma Interactiva

Bienvenido a **Tecnoteca**, una plataforma interactiva educativa diseñada para ofrecer una experiencia divertida y educativa.

## 📋 Características

- 🎮 **Quiz Interactivo**: Juego de preguntas sobre tecnología y educación
- 📋 **Encuesta de Feedback**: Sistema para recopilar opiniones de usuarios
- 🎨 **Diseño Responsivo**: Compatible con dispositivos de escritorio y móviles
- ✨ **Interfaz Moderna**: Animations y efectos visuales atractivos

## 🗂️ Estructura del Proyecto

```
.
├── index.html           # Página de inicio
├── index.css            # Estilos principales
├── images/              # Imágenes y recursos visuales
├── fonts/               # Fuentes personalizadas
│   ├── bouncy_2/
│   ├── gwenchana/
│   └── kg_happy/
├── pages/               # Páginas internas
│   ├── quiz.html        # Página del quiz
│   ├── survey.html      # Página de encuesta
│   └── page-theme.js    # Estilos dinámicos
└── .gitignore           # Archivos ignorados por Git
```

## 🚀 Cómo Usar

1. Clona este repositorio:
```bash
git clone https://github.com/usuario/tecnoteca.git
```

2. Abre `index.html` en tu navegador web

3. ¡Ingresa tu nombre y comienza a explorar!

## 💻 Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conexión a Internet (para cargar fuentes de Google)

## 📝 Notas Importantes

- Las fuentes personalizadas están alojadas en las carpetas `fonts/`
- Los recursos de imagen están en la carpeta `images/`
- Los datos del quiz se almacenan localmente en el navegador
- La encuesta puede conectarse con Google Apps Script para guardar respuestas

## 🔗 Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Estilos avanzados con gradientes y animaciones
- **JavaScript**: Interactividad y lógica de la aplicación
- **Google Fonts**: Tipografía externa
